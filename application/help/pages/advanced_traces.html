Advanced Traces

The trace function has a number of parameters that can help you as a developer to optimize your trace output. The full syntax of the trace function is: 

MonsterDebugger.trace(target, object, color, functions, depth);

Argument

 Type

 Description
target Object The target from where the trace is called, this usually is the keyword "this".
object * The object to trace.
color uint The color of the trace in the desktop application. For instance if you're working in a team, each person could have their own color.

functions Boolean Display the public methods of a Class in the output panel.
depth int The depth of the trace. By default the trace will be 4 levels deep. If you want to trace a complex XML document, that has a structure deeper than 4 children, you should change this value. For unlimited depth set the value to -1. WARNING: This can seriously stall your app!


Example of a more complex trace

var loader:Loader = new Loader();
MonsterDebugger.trace(this, loader, 0xFF0000, true, 6);
