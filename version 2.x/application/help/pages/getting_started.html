Getting Started

De MonsterDebugger consist of two parts: A desktop application and a client side Actionscript 3.0 class. The client side class can be used in both Flash and Flex projects, with minimal differences. The debugger client class should be initialized once at the highest level of your application (in Flash this is the document class, in Flex this is the base MXML document). You can send traces from any point in you application by calling the static Debugger.trace() function.
