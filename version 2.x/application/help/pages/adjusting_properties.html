Adjusting Properties

When you click on an object in the live application panel, the inspector panel will display its properties and methods. To edit a property, just click on its value. Any changes to a property are immediately applied to the object. You can see the results straight away in your own application. Note: You can edit properties of the type String, Number, Boolean, int or uint at runtime.
